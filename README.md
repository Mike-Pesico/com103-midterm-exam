## 📖 Overview
The **Task Tracker** is a Python-based console program designed to simulate a simple project management system for group activities. It tracks task assignments, monitors progress, and calculates overall project completion using a point-based system.

This project demonstrates foundational programming concepts such as lists, loops, and conditional logic.

---

## 🚀 Features
- Project title and group name input  
- Predefined task categories with estimated hours  
- Assignment of tasks to group members  
- Optional task skipping feature  
- Automatic scoring system:
  - Done = 2 points  
  - Pending = 1 point  
- Progress computation in percentage  
- Automatic project status classification  
- Final formatted task board output

## ⚙️ How It Works

1. Enter project title and group name  
2. View available task types  
3. Assign up to 4 tasks  
4. Input member name and status  
5. System calculates:
   - Total points  
   - Maximum possible points  
   - Progress percentage  
6. Displays final project dashboard  

---

## 📊 Grading Logic

Done = 2 points  
Pending = 1 point  

Progress = (Total Points / Maximum Points) × 100  

Project Status:
- ≥ 75% → PROJECT READY  
- ≥ 50% → ON TRACK  
- < 50% → NEEDS MORE WORK
